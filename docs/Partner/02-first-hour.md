# 02 - First Hour

Navigation: Previous `01-your-role.md` | Index `README.md` | Next `03-daily-partnership-loop.md`

Your first hour should produce one visible loop: trigger -> plan -> action -> traceable outcome.

## Step 1 - Pick One Sandbox Task

Use one issue in the sandbox scope (`Vaquum/Agent1` with sandbox markers).

Keep it small and concrete.

## Step 2 - Trigger Agent1 Naturally

Mention Agent1 in context, or assign the issue if context is complete.

If context is incomplete, Agent1 should ask for clarification instead of guessing.

## Step 3 - Observe The Dashboard

Track:

- job creation
- state transitions
- side-effect attempts
- timeline details and trace correlation

## Step 4 - Close The Loop

Confirm that the issue reaches the expected next state (human feedback, completion, or explicit block with reason).

What matters here is not speed. It is trust in deterministic behavior.

## Success Criteria For Hour One

- You can explain what happened and why.
- You can find the exact trace and event history.
- You can tell where human choice still governs outcome.
